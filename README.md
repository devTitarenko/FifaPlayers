For launching
-------------------------
Fill in all fields in `src/main/resources/properties/app.properties`.

To initialize the database, do deploy of the project.

It is necessary to install a Lombok plugin!


Technologies
-------------------------
- Java 8
- Oracle Relational model
- Liquibase
- Hibernate 4
- Spring Framework 4
- JSF
- Primefaces
- Maven
- Lombok
- Tomcat
